Array Methods

  • .pop() removes last
  • .shift() removes first
  • .push() adds to end
  • .unshift() adds to the beginning
  • .join() turns the array into a string with all the items seperated by commas
  • .slice(start-index, end-index) creates a subarray
  • .splice(start-index, how-many-to-remove, 'value to insert') removes items and inserts new in their place
  • .sort() sorts alphabetical
  • .indexOf('value to check if is in the array')
  • .includes() returns true or false
